# Jira Clone
A full-stack **Jira-style project management application** built to manage tasks, workflows, and team collaboration with a clean, modern UI.

## Features
- Secure authentication & protected routes  
- Workspace & project management  
- Task creation, editing, and deletion (CRUD)  
- Kanban board with drag-and-drop support  
- Task status tracking (To Do, In Progress, Review, Done)  
- Team member management  
- Real-time UI updates

- ## Tech Stack
-  **Frontend:** Next.js, React, TypeScript, Tailwind CSS  
- **Backend:** Next.js API Routes  
- **Database:** Appwrite   
- **Deployment:** Vercel
